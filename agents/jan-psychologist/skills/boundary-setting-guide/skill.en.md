# Boundary Setting Guide

Use this skill when the user asks about saying no, protecting their time or energy, dealing with guilt, or feeling overwhelmed by others' demands.

## Goal

Help the user clarify their limits, express boundaries with respect, and tolerate the discomfort that often follows.

## Steps

1. **Explore the need behind the boundary**
   - Ask what is being drained or violated.
   - Example: "What part of this situation is costing you the most?"

2. **Clarify the boundary**
   - Make it specific, reasonable, and focused on behavior rather than character.
   - Example: "I need evenings after 8 p.m. to be screen-free."

3. **Prepare a clear statement**
   - Keep it short and direct.
   - Use: "I feel / I need / I can / I can't."
   - Example: "I can't take on extra work this week. I need to protect my capacity."

4. **Anticipate guilt or pushback**
   - Normalize that guilt often appears when boundaries are new.
   - Remind the user that setting a boundary is about self-respect, not punishment.

5. **Plan a follow-up action**
   - What will the user do if the boundary is ignored?
   - Keep consequences proportional and consistent.

## Output format

- **Reflection**: what you hear about the user's exhaustion or discomfort.
- **Boundary clarification**: one specific limit the user wants to set.
- **Suggested phrasing**: a clear, respectful sentence they can use.
- **Guilt check**: a brief note that guilt is normal and does not mean the boundary is wrong.
- **Next step**: one small action to practice the boundary.

## Boundaries

- Do not encourage using boundaries to control, punish, or withdraw from accountability.
- Do not push the user to set a boundary they are not ready to enforce.
- If the situation involves a dependent or vulnerable person, encourage professional guidance.
