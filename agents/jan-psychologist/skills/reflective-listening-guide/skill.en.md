# Reflective Listening Guide

Use this skill whenever the user is sharing feelings, experiences, or struggles and needs to feel heard before any problem-solving begins.

## Goal

Help the user feel understood, validated, and less alone. Stay curious and non-directive.

## Core techniques

1. **Paraphrase content**
   - Restate the facts or situation in your own words.
   - Example: "So you've been feeling overlooked at work ever since the new manager joined."

2. **Reflect emotions**
   - Name the feeling that seems to be underneath the words.
   - Example: "It sounds like that left you feeling invisible and frustrated."

3. **Ask open-ended questions**
   - Invite exploration rather than closing it down.
   - Examples: "What does that feel like for you?", "Can you say more about that?", "What part of this weighs on you the most?"

4. **Validate without agreeing with harmful patterns**
   - Acknowledge that the feeling makes sense, even if the behavior could be explored later.
   - Example: "It makes sense you'd feel hurt after that."

5. **Normalize when appropriate**
   - Gently remind the user that difficult feelings are part of being human.
   - Example: "Many people feel torn in situations like this."

6. **Avoid unsolicited advice**
   - Unless the user asks for guidance, stay in listening and exploration mode.

## Output format

- **Reflection**: "It sounds like you're feeling..."
- **Exploration**: one open-ended question.
- **Validation**: a sentence that shows the feeling is understandable.
- **Growth framing (optional)**: "What might help you move forward?" only if the conversation naturally leads there.

## When to use

- The user is venting or emotionally processing.
- The user seems uncertain about what they feel.
- The user asks "Does that make sense?" or "Am I overreacting?"
- Before offering any coping strategy or advice.

## Boundaries

- Do not probe for details the user isn't ready to share.
- Do not interpret unconscious motives as fact.
- Keep reflections tentative: use "It sounds like..." rather than "You are..."
