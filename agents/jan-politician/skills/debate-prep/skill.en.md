# Debate Prep

Use this skill when the user wants to prepare for, understand, or simulate a political debate on a contested issue.

## Goal

Help the user see the strongest arguments and likely rebuttals on each side of a political question, grounded in facts rather than rhetoric.

## Steps

1. **Clarify the motion or question** — state the exact proposition being debated.
2. **Identify the competing frames** — summarize how each side defines the problem and the values at stake.
3. **List core arguments for each side** — present the strongest 3–5 points, ordered by impact.
4. **Anticipate rebuttals** — for each argument, note the most common counterarguments and their factual basis.
5. **Ground claims in evidence** — cite statistics, historical examples, expert consensus, or authoritative sources.
6. **Flag logical fallacies or weak evidence** — point out arguments that rely on anecdote, strawmen, or unsupported assertions.
7. **Suggest strategic notes** — indicate which clashes are decisive and where nuance is most needed.

## Output format

- **Motion**: the precise question or proposition.
- **Framing**: how each side sees the issue.
- **Arguments for**: strongest points with evidence.
- **Arguments against**: strongest points with evidence.
- **Likely rebuttals**: counters for each main argument.
- **Factual grounding**: key statistics, studies, or precedents.
- **Risk areas**: weak or overclaimed points on either side.
- **Sources**: citations for factual claims.

## Boundaries

- Do not write campaign speeches or attack ads.
- Do not strawman any position; present the strongest version of each side.
- Avoid inflammatory labels when naming positions.
