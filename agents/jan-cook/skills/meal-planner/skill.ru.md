# Планировщик питания

Используй этот навык, когда пользователь хочет недельное меню, план заготовок, список покупок или помощь в согласовании приёмов пищи с диетическими предпочтениями и временными ограничениями.

## Цель

Составить практичное, сбалансированное недельное меню с графиком подготовки и организованным списком покупок.

## Шаги

1. **Собери ограничения** — Диетические ограничения, аллергии, предпочтения по кухне, бюджет, доступное время, кулинарный уровень, размер семьи, оборудование.
2. **Спланируй меню** — Распредели блюда по дням/приёмам пищи; соблюдай разнообразие, питательность и повторное использование ингредиентов.
3. **Ориентируйся на эффективность** — Готовь заготовки партиями, повторно используй белки, распределяй подготовку по доступному времени.
4. **Составь график подготовки** — Разбей задачи по дням, чтобы избежать авралов.
5. **Сформируй список покупок** — Сгруппируй товары по отделам магазина; укажи количества и замены.
6. **Добавь примечания** — Советы по хранению, разогреву и примерную стоимость, если уместно.

## Формат вывода

- **Недельное меню**: приёмы пищи по дням с краткими названиями блюд.
- **График подготовки**: задачи, распределённые по неделе.
- **Список покупок**: сгруппирован по категориям с количествами.
- **Примечания**: хранение, разогрев, масштабирование, замены.

## Границы

- Давай только общие рекомендации по питанию; не назначай терапевтические диеты и не давай медицинских советов.
- Учитывай диетические ограничения без осуждения.
- Не делай предположений о бюджете или наличии товаров; спрашивай, когда это важно.
