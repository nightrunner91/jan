# Гид по замене ингредиентов

Используй этот навык, когда у пользователя нет ингредиента, есть диетическое ограничение или он хочет заменить ингредиент по вкусу, текстуре или доступности.

## Цель

Рекомендовать лучшую замену, чётко объясняя влияние на вкус, текстуру, структуру и безопасность.

## Шаги

1. **Определи роль ингредиента** — Он даёт жир, жидкость, кислоту, структуру, разрыхление, связку, вкус или текстуру?
2. **Перечисли возможные замены** — Предлагай варианты по категориям (без молочных, без глютена, веганские, с низким содержанием жира и т.д.).
3. **Оцени каждую замену** — Лучшее совпадение, приемлемое, только в крайнем случае.
4. **Объясни влияние** — Для каждого варианта отметь изменения вкуса, текстуры, структуры и безопасности пищи.
5. **Укажи соотношения** — Приведи меры замены, где применимо.
6. **Предупреди об опасных заменах** — Предупреждай о заменах, создающих риски безопасности или аллергенов.

## Формат вывода

- **Исходный ингредиент**: что требуется в рецепте и его роль.
- **Лучшие замены**: ранжированный список с соотношениями.
- **Примечания по влиянию**: вкус / текстура / структура / безопасность для каждой.
- **Когда не стоит заменять**: замены, которые кардинально изменят или опасны для блюда.

## Границы

- Не предлагай замены, создающие угрозу безопасности пищи (например, исключение кислоты в рецепте консервирования).
- Спроси об аллергенах, прежде чем предлагать распространённые аллергены в качестве замены.
- Оставайся в рамках общих принципов питания; избегай медицинских заявлений.
