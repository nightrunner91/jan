# Safety & Code Checker

Use this skill to review the user's project for hazards, code issues, and situations that require professional oversight.

## Goal

Keep the user safe and compliant by flagging risks and boundaries between DIY and licensed work.

## Steps

1. **Identify the work type**
   - Renovation, new build, repair, electrical, plumbing, structural, roofing, etc.

2. **Check for professional-only work**
   Flag any of the following and explicitly recommend a licensed professional:
   - Load-bearing wall removal or modification.
   - Foundation, beam, column, or footing changes.
   - Electrical panel, main service, or new circuit installation.
   - Gas line installation or modification.
   - Plumbing changes affecting main sewer lines or pressurized supply.
   - Roofing structural repairs.
   - Asbestos, lead paint, or mold remediation.

3. **Assess hazards**
   | Severity | Meaning | Example |
   |---|---|---|
   | Critical | Stop work immediately | Exposed live wiring, unstable structure |
   | High | Requires mitigation before continuing | No fall protection on a roof, silica dust |
   | Medium | Manage with PPE/procedure | Sharp tools, chemical adhesives |
   | Low | Good practice reminder | Clean work area, proper lifting |

4. **Reference codes when applicable**
   - Use building codes (e.g., IRC, IBC, NEC, local codes) or regional standards (e.g., GOST for Russian-speaking regions).
   - Always cite the standard and note that local amendments may apply.
   - Prefer general guidance; do not issue code compliance certifications.

5. **Summarize**
   - List what is safe to DIY.
   - List what requires a professional.
   - List required PPE and permits.

## Boundaries

- Do not provide structural engineering calculations.
- Do not diagnose hazardous materials visually from photos alone.
- Always recommend professional inspection when uncertainty exists.
