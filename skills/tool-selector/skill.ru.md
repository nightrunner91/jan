# Подбор инструментов

Используй этот навык, когда пользователю нужны рекомендации по инструментам для строительных, ремонтных или DIY-работ.

## Цель

Рекомендовать подходящие инструменты для задачи, объяснить безопасное использование и дать советы по покупке, аренде или одалживанию.

## Шаги

1. **Пойми задачу**
   - Какая операция требуется? (резка, сверление, крепление, измерение, отделка и т.д.)
   - С какими материалами ведётся работа? (дерево, металл, бетон, плитка, гипсокартон и т.д.)
   - Как часто инструмент будет использоваться? (разово, иногда, часто)
   - Какой бюджет и опыт у пользователя?

2. **Рекомендуй необходимые инструменты**
   - Сначала перечисли обязательные инструменты.
   - Затем — полезные, но необязательные.
   - Различай ручной и электроинструмент.

3. **Дай рекомендации по выбору**
   | Фактор | На что обратить внимание |
   |---|---|
   | Сетевой vs. аккумуляторный | Время работы, мощность, мобильность |
   | Класс/бренд | Разовый DIY или профессиональное использование |
   | Эргономика | Вес, рукоятка, вибрация |
   | Средства защиты | Защитные кожухи, пылеотвод, защита от отдачи |
   | Расходники | Диски, свёрла, батареи, абразивы |

4. **Объясни безопасную настройку и работу**
   - Необходимые СИЗ.
   - Предварительный осмотр.
   - Надёжное закрепление заготовки.
   - Типичные ошибки, приводящие к травмам или плохому результату.

5. **Дай базовые советы по обслуживанию**
   - Очистка после работы.
   - Признаки необходимости замены диска/сверла.
   - Уход за аккумуляторами.

6. **Совет по покупке**
   - Покупай, если используешь несколько раз в год.
   - Арендуй для крупных редких инструментов (например, плиткорез, отбойный молоток).
   - Одолжай для узкоспециализированных инструментов, нужных один раз.

## Примечание по безопасности

Никогда не рекомендуй снимать защитные кожухи или использовать инструмент не по назначению.
