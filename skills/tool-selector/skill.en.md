# Tool Selector

Use this skill when the user needs tool recommendations for a construction, renovation, or DIY task.

## Goal

Recommend the right tools for the job, explain safe use, and advise on buying, renting, or borrowing.

## Steps

1. **Understand the task**
   - What operation needs to be performed? (cut, drill, fasten, measure, finish, etc.)
   - What materials are involved? (wood, metal, concrete, tile, drywall, etc.)
   - How often will the tool be used? (one-time, occasional, frequent)
   - What is the user's budget and experience level?

2. **Recommend required tools**
   - List essential tools first.
   - Then list helpful but optional tools.
   - Distinguish hand tools from power tools.

3. **Provide selection guidance**
   | Factor | What to consider |
   |---|---|
   | Corded vs. cordless | Runtime, power needs, portability |
   | Brand/tier | Occasional DIY vs. professional use |
   | Ergonomics | Weight, grip, vibration |
   | Safety features | Blade guards, dust extraction, kickback protection |
   | Consumables | Blades, bits, batteries, abrasives |

4. **Explain safe setup and operation**
   - PPE required.
   - Pre-use inspection.
   - Stable workpiece setup.
   - Common mistakes that cause injury or poor results.

5. **Give maintenance basics**
   - Cleaning after use.
   - Blade/bit replacement signs.
   - Battery care for cordless tools.

6. **Buying advice**
   - Buy if used more than a few times per year.
   - Rent for large, infrequent tools (e.g., tile saw, jackhammer).
   - Borrow for specialty tools used once.

## Safety note

Never recommend bypassing safety guards or using a tool for a purpose it was not designed for.
