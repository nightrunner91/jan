# Build Planner

Use this skill to structure any construction, renovation, or DIY project into a clear, repeatable plan.

## Goal

Translate the user's project into an actionable sequence that matches the Constructor's standard output format.

## Required output format

Every project plan should include:

1. **Overview**
   - What is being built and why.
   - Scope summary.

2. **Materials & tools needed**
   - Itemized materials with quantities and units.
   - Tools required (owned + to borrow/buy/rent).
   - Optional: estimated total material cost.

3. **Step-by-step procedure**
   - Numbered steps in logical order.
   - One action per step when possible.
   - Include measurements, drying/curing times, and inspection points.

4. **Approximate time & skill level**
   - Total estimated hours/days.
   - Skill level: Beginner / Intermediate / Advanced.

5. **Safety notes**
   - Required PPE.
   - Critical hazards.
   - When to stop and call a professional.

## Planning tips

- Break large projects into phases (prep → rough work → finish → cleanup).
- Identify dependencies (e.g., framing before drywall, primer before paint).
- Build in buffer time for drying, curing, and unexpected issues.
- For each step, ask: "What could go wrong here, and how do I prevent it?"

## Safety rule

Never omit the safety section. If a step involves electricity, gas, heights, or load-bearing changes, add an explicit professional-consultation warning.
